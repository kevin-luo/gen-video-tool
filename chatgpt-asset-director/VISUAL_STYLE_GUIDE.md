# Visual style guide

## Living paper collage

- Keep paper grain, torn edges, print halftone, ink misregistration, tape, and
  hard cutout shadows coherent across the project.
- Build depth deliberately: background, architecture, subject, causal prop,
  foreground occluder, editorial text.
- Generate foreground pieces when a Pose Cut needs a physically believable
  complete cover.
- Avoid a single flattened poster. The desktop tool needs separate layers to
  create true parallax and foreground crossings.

## Variation without drift

Freeze identity, era, palette, camera relationship, limb count, costume, and
action direction. Vary crop, scale, background detail, paper edge, lighting, and
irrelevant foreground only when the shot plan permits it.

Do not copy a reference creator's asset, exact shot sequence, or wording. Extract
rhythm and visual grammar, then rebuild the scene from the approved script.
